Cash Flow Visualization Tool

Welcome to the Cash Flow Visualization Tool for Companies! This application allows you to upload your transaction data and visually analyze your cash flows using an intuitive graph representation. Each node represents an account, and the edges represent financial transactions.

With our tool, you can:

  • Visualize cash flows between accounts
  • Identify high transaction frequencies
  • Analyze transfers between customers and suppliers
  • Filter data by transaction type and currency
